I.
1.Install Nodejs from google -->to check version node -v
2.Install Angular and CLI -->npm install -g @angular/cli & to check version -->ng version
3.Create Application -->ng new app_name
4.Move to the Application -->cd app_name
5.Run Server -->ng serve
6.Create Routing -->ng generate module app-routing --flat --module=app
  Routing:
	i.add this line into head tag--> <base href="/">
	ii.add RoutingModule and forRoot() method into imports of app.module.ts file and create Routes array.
	iii.add router-outlet tag where you want to display your content
7.To open our project into vs code --> code .

III.
1.Create new Components -->ng generate component login (ng g c login)
	creating sub-component --> ng g c login/admin-login
2.Create module(all the components present inside the module)-->ng generate module userauth
3.Create component inside the modue -->ng g c userauth/gallery
4.Create Class file-->ng g class class-name (generate 2 files,main file and testing file)
5.Create service file-->ng g service api-service
6.Build project -->ng build
