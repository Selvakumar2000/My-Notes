Description:
--------------
1.Create new project >ng new DatingPortal
2.Would you like to add Angular routing? (y/N) y
3. Which stylesheet format would you like to use? (Use arrow keys)
> CSS
4.Create HomePage component >ng g c homepage
5.Create User component  >ng g c user
  #Create child-component of user
	 i.signup >ng g c user/signup
	ii.login  >ng g c user/login
	
6.Create User-Details component >ng g c user-details
  #Create child-component of user-details
	 i. details-form  >ng g c user-details/details-form

7.Create Suggestions component >ng g c suggestions
 #Create child-component of suggestions
	 i.display-allDetails >ng g c suggestions/display-allDetails

8.Create service file for user-details component -> ng g s shared/user-details
  Create a model file for user-details -> ng g class shared/user-details --type=model
  Create service file for user -> ng g s shared/user
